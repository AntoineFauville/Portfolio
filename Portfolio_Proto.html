Level Design Concept Maps

Classic

  • Level Design Maps are planning or concept arts I made for either fun or for projects. Whenever I have time or I travel I draw sketches of maps.
    I always pitch myself with a game idea, then I try to create a map that could work for it.

    I usually first think about a small documentation of the maps, including the intentions and the goal of the map.
    Then I implement the map using assets or create mine to test my assumption and iterate on the map. Always trying to project the player experience before anything else.

    • Planet Coaster Maps

    Hobby time

  • Made with Planet Coaster Engine
  • Planet Coaster is a simulation type of game about theme parks (which i'm passionate about).
    Like the predecessor, the game allow you to create roller coaster, attractions, but in this one, you can also decorate your park like never before. It's pretty similar with a game engine or a 3D modeling tool.
  • As a hobby, after school or after work, I like to spend time on games like this. I've made few park, here is an exemple in the pictures, thinking and planning it before making it. I stayed on a theme with a color pallet.
    Using basic environment design principles. I've also thought about User Ergonomy in the park, with all the needs, but also all the "hiding" (Never show that a line is full so people pay and get into the attractions). I love spending my time on making these parks.

    • Industrial Entreprise

    2015

  • Made with Unity for ESA ST-Luc
  • Industrial Enterprise This was a small project made for a school demand. Second year project at Saint-Luc ESA in Saint-Gilles, Brussels. Digital Arts Studies.
    Assignment :
    - Theme : Utopia.
    - Build a flat with an ambiance and some personal esthetic.
    - At least 3 interactions.
    - made in Unity and in 3D.
    I did add an extra room if you activate everything, this room was another exercise. Tribute to an art piece. There is no goal.
  • My intention here was to create a space and ask questions about this space What we are going to be living in, in 2030 ? This is a dystopia where everyone has the same flat, everyone working at the same place (It’s a critic of some actual social system).
