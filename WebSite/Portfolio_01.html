Bombslinger :

A Bomberman-inspired Wild West action game with bombs!
A solo hardcore reinvention of the iconic explosive action! I have worked on map design, i did around 600+ maps for them.
I’ve worked on general game design, i’ve done some google sheets with designs like procedural generation for monsters, iterations on start rooms or other analytics sheets.
Those sheets served me later when they asked me to do some balancing ( monsters presets, gold, mana, hardness of the game, general feeling).
I’ve done some QA testing, using “Mantis” as bug repporter. I’ve reworked some art assets and worked on some UI and UX design.
