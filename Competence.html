Level Design


Designed maps and got interested into creating maps since I was young, starting with age of empire 2, and 3 were I created scenarios for solo, or some multiplayer content. Stronghold 2 was a big start as well.
I got hooked in Roller Coaster Serie, Minecraft or other games where I could create an environment. Doom is also on my checklist, when it comes to create maps for me and my friends.



It got serious when I started developing games, I always pitch myself with a game idea, then I want to create a map that could relate to the game or work for it. 2D maps or 3D maps. I always try to create my maps thinking about the game intentions and goals.

As a passionate level designer, I love using level building tools and I have a pretty good understanding of what a game / level production pipeline is.
