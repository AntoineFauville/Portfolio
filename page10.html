Planet Coaster 

Planet Coaster Engine

Planet Coaster is a simulation type of game about theme parks (which i'm passionate about).
Like the predecessor, the game allow you to create roller coaster, attractions, but in this one, you can also decorate your park like never before. It's pretty similar with a game engine or a 3D modeling tool.

As a hobby, after school or after work, I like to spend time on games like this. I've made few park, here is an exemple in the pictures, thinking and planning it before making it. I stayed on a theme with a color pallet.
Using basic environment design principles.

I've also thought about User Ergonomy in the park, with all the needs, but also all the "hiding" (Never show that a line is full so people pay and get into the attractions). I love spending my time on making these parks.
